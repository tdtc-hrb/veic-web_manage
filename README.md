veic_web Pages
========

使用向导建立工程:
  ASP.NET Core Web App

# development environment
- SDK: .NET Core 3.1+    
- IDE: VS2019+    
- Database: MySQL 5.7+    
- Web Server: IIS 10.0+

# project structure
- Razor Pages    
- Front end: Bootstrap v5    
- JS library: JQuery3.5+


# About Razor Page(CURD)
> EF Core 5.0 兼容MySql.Data.EntityFrameworkCore 8.0.23+.

在Pages新建文件夹Articles, 点击
```bash
Add -> Razor Page -> Razor Pages using Entity Framework(CURD)
```
